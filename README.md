# 地下軟體世界 自動下單 API (非官方)

## 安裝 python3 & pip3
```shell
sudo apt install python3
sudo apt install python3-pip
```
## 安裝 python-requests & 2captcha-python
```shell
sudo pip3 install requests
sudo pip3 install 2captcha-python
```
## 使用2recaptcha服務
官方網站：https://2captcha.com/

此專案只作為技術研討之用途，不作其他用途。
